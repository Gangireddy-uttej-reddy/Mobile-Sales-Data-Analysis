# Mobile-Sales-Data-Analysis
Mobile Sales Data Analysis is a Python-based project that analyzes mobile sales data to extract meaningful insights. It uses NumPy to process and clean data, calculate key metrics, and identify trends in sales performance. The project helps in understanding customer behavior and supports better business decision-making.
# 📊 Mobile Sales Data Analysis

## 📌 Description

Mobile Sales Data Analysis is a Python-based project that focuses on analyzing mobile sales data to extract meaningful insights. It processes raw data, performs calculations, and identifies trends that help in understanding sales performance and customer behavior.

## 🚀 Features

* Data cleaning and preprocessing
* Sales performance analysis
* Trend identification
* Numerical computations using NumPy
* Insight generation for decision-making

## 🛠️ Technologies Used

* Python
* NumPy

## ▶️ How to Run

1. Install Python
2. Install required libraries (NumPy)

   ```bash
   pip install numpy
   ```
3. Open and run the `.ipynb` file using Jupyter Notebook
4. Execute all cells to view analysis results

## 📊 Use Case

Helps businesses analyze mobile sales data, identify trends, and make data-driven decisions.

## 📂 Project Structure

* `Mobile Sales data Analysis.ipynb` – Main project file containing analysis
